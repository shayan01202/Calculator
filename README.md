
# 🧮 Java Calculator

A simple console-based calculator built using Java. This program performs basic arithmetic operations like addition, subtraction, multiplication, and division.

---

## 📌 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division (with zero division handling)
* User-friendly console interface

---

## 🛠️ Technologies Used

* Java
* Scanner Class (for user input)

---

## 🚀 How to Run

### 1. Compile the program

```bash
javac Calculator.java
```

### 2. Run the program

```bash
java Calculator
```

---

## 📋 Sample Output

```
===== JAVA CALCULATOR =====
Enter first number: 10
Enter second number: 5

Choose operation:
1. Addition
2. Subtraction
3. Multiplication
4. Division

Result = 15.0
```

---

## ⚠️ Error Handling

* Displays an error message when division by zero is attempted.
